### ブロックとは？

- ブロックは、コマンドと出力を1つの原子単位にまとめたもの

- ブロックは、私たちが簡単にできるようにします：
  - コマンドをコピーする
  - コマンドの出力をコピーする
  - コマンドの出力開始位置まで直接スクロールできる
  - コマンドの再入力
  - コマンドとその出力の両方を共有する（フォーマット付き！）。
  - ブックマークコマンド
